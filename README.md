Study Planner SaaS (WIP)

This project is a Software as a Service (SaaS) under development for planning, executing, and tracking study routines for Brazilian public service entrance exams.

The main goal is not only the final product, but to master the complete software development lifecycle, from frontend to production deployment, using a modern technology stack widely adopted in enterprise environments.

## Project Goals

Develop a cycle-based study planning system

Build a “living digital spreadsheet” with business rules and automated feedback

Work in incremental development cycles, delivering minimal value at each stage

Master the full software lifecycle:

Frontend

Backend

Database

Docker

Deployment

CI/CD

Kubernetes (lab environment)
